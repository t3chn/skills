# Notion Integration

URL: https://www.greptile.com/docs/notion-integration

Connect Notion to give Greptile access to your documentation, specs, and business context during code reviews.

## What You Get

* **Spec document analysis** - Greptile reads product requirements and technical specifications
* **Architecture alignment** - Verifies code follows documented architecture patterns
* **Business context** - Understands product goals and constraints from planning docs
* **Design system compliance** - Checks against design system documentation

## Setup

Go to Integrations

Click **Integrations** in the left sidebar of your Greptile dashboard.

Connect Notion

Click **Connect** on the Notion card.

Get Notion API Key

```
1. Go to the [Notion integrations page](https://www.notion.so/profile/integrations)
```
2. Create a **New Integration** (or use existing)
* Enter integration name
* Select associated workspace
* Choose integration type
3. Go to **Configuration** tab
4. Set capabilities (Read, Update, Insert content)
5. Copy the **Internal Integration Secret**

Enter your API key and click **Connect Notion** .

Done

Notion appears as a connected data source in your dashboard.
